# ttk-tools

### 安装
```
sudo npm i -g ttk-tool
```
###创建website


```
ttk w myproject 
```
或者
```
ttk website myproject
```

### clone app

```
ttk c ttk-edf-app-login[/*npm 上已经发布过的app项目*/] apps/edf/ttk-edf-app-login [/*clone app 到指定的路径*/]
// 或者
ttk clone ttk-edf-app-login apps/edf/ttk-edf-app-login
```
